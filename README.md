# ArduinoLed

Implemented LED lights which reacts to the music being played through ARDUINO.

Led Type - WS2811
Processor - Arduino

Requirements:


Led strips(WS2811)
Processor (Arduino)
Resistors(47k,1M)
Capacitors(1uf,4.7uf)
Op-Amps(324)
